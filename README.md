# on-yiwan28-guide

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类独立编写的 HTML 文件，这些页面通常用于记录、展示或作为临时工具使用。仓库本身不绑定任何特定网站或服务，所有页面均以静态 HTML 形式存在，方便直接浏览或部署。

如果你需要快速查看某个页面的内容，可以通过 GitHub Pages 或其他静态托管服务直接访问仓库中的 HTML 文件。

## 目录说明

仓库的根目录下直接放置 HTML 文件，每个文件通常代表一个独立的页面。文件命名尽量简洁，并带有一定的描述性，方便识别用途。

```
/
├── index.html          (仓库主页说明，可选)
├── example-page.html   (示例页面)
├── tool-page.html      (工具类页面)
└── ...
```

如有子目录，则用于存放与该页面相关的资源文件（如 CSS、JavaScript、图片等），但大部分页面尽量保持自包含，减少外部依赖。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面内容可能包含实验性功能、临时记录或学习笔记，不保证长期可用或兼容所有浏览器。
- 部分页面可能引用外部资源（如字体、图标库、API），这些资源由第三方提供，本仓库不对此负责。
- 页面更新频率不定，以实际归档需要为准。

## 维护说明

本仓库由个人维护，主要用于归档和分享。如有问题或建议，欢迎提交 Issue 或 Pull Request，但请注意：

- 不保证及时回复或处理所有请求。
- 不接受包含商业推广、广告或恶意内容的提交。
- 对于页面内容的修改，请尽量说明原因。

## 许可

本仓库中的 HTML 文件默认采用 [MIT 许可证](LICENSE)（如果仓库中包含 LICENSE 文件）。每个页面也可在文件头部标注单独的许可信息，以该标注为准。
